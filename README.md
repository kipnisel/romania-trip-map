# 🗺️ מפת טיול רומניה — עמוד שיתוף עם ביקורות

עמוד אינטרנט (GitHub Pages) עם מפה אינטראקטיבית של כל מוקדי הטיול, שכל בן משפחה יכול לפתוח, לקרוא ולכתוב ביקורות. הביקורות נאספות אוטומטית ל-Google Sheet אחד.

הקבצים:
- `index.html` — העמוד (מפה + מערכת ביקורות)
- `apps-script.gs` — קוד ה-Backend (Google Apps Script)
- `README.md` — ההוראות האלה

---

## שלב 1 · הקמת ה-Backend (Google Sheet + Apps Script) — ~5 דקות

1. היכנס ל-[sheets.new](https://sheets.new) וצור **גיליון חדש** (למשל בשם "ביקורות רומניה").
2. בתפריט: **Extensions ▸ Apps Script**.
3. מחק את הקוד הקיים, והדבק את **כל התוכן** של `apps-script.gs`. שמור (💾).
4. לחץ **Deploy ▸ New deployment**.
5. ליד "Select type" לחץ על גלגל השיניים ⚙️ ובחר **Web app**.
6. הגדר:
   - **Execute as:** `Me`
   - **Who has access:** `Anyone`
7. לחץ **Deploy**, אשר הרשאות (בחר את חשבון הגוגל שלך → Advanced → Allow).
8. העתק את ה-**Web app URL** — כתובת שמסתיימת ב-`/exec`.

## שלב 2 · חיבור העמוד ל-Backend

1. פתח את `index.html` בעורך טקסט.
2. מצא בראש ה-`<script>` את השורה:
   ```js
   const SCRIPT_URL = "PASTE_YOUR_APPS_SCRIPT_URL_HERE";
   ```
3. החלף בכתובת שהעתקת:
   ```js
   const SCRIPT_URL = "https://script.google.com/macros/s/XXXXX/exec";
   ```
4. שמור.

## שלב 3 · פרסום ב-GitHub Pages — ~3 דקות

> ⚠️ **אל תשתמש ב-repo האישי `MyDocs`** (יש בו קבצים פרטיים). צור repo חדש ונפרד.

1. היכנס ל-[github.com/new](https://github.com/new) וצור repo **ציבורי (Public)**, למשל `romania-trip-map`.
2. במסך ה-repo: **Add file ▸ Upload files**, וגרור את `index.html` (ואם תרצה גם README).
   *(את `apps-script.gs` אין צורך להעלות — הוא כבר הודבק בגוגל.)*
3. **Commit changes**.
4. **Settings ▸ Pages** → תחת "Branch" בחר `main` ותיקייה `/ (root)` → **Save**.
5. אחרי 1–2 דקות העמוד יהיה זמין בכתובת:
   ```
   https://<שם-המשתמש>.github.io/romania-trip-map/
   ```
6. **שתף את הקישור** עם בני המשפחה — כל אחד יכול לכתוב ביקורות בלי חשבון.

## איך זה עובד

- **מפה:** Leaflet + OpenStreetMap (בלי מפתח API). 33 מוקדים מקודדים בצבע לפי אזור, כולל הדירות ושדה התעופה.
- **ביקורות משותפות לכולם:** כל מבקר רואה את ביקורות כל מי שכתב לפניו. הביקורות נשמרות ב-Google Sheet ונטענות מחדש (טעינה טרייה, ללא cache) בכל כניסה ובכל פתיחת מוקד — כך שהמבקר הבא תמיד רואה את המצב המעודכן.
- **חיווי על המפה:** מוקד שכבר יש עליו ביקורות משפחה מסומן בטבעת זהובה, והחלונית מציגה "👨‍👩‍👧 N ביקורות משפחה · ★ ממוצע".
- **ביקורת על מוקד:** לוחצים על נקודה במפה → "📝 ביקורות משפחה" → קוראים את הקודמות ומוסיפים (שם, דירוג כוכבים, הערה).
- **כל הביקורות:** כפתור בראש העמוד מרכז את כל הביקורות מכל המוקדים.
- **איסוף מרכזי:** כל ביקורת נשמרת כשורה ב-Google Sheet שלך (עמודות: זמן, מוקד, שם, דירוג, הערה). אפשר לסנן/למיין/לייצא משם.

## עדכון תוכן
- להוספת/שינוי מוקדים — ערוך את מערך `POIS` ב-`index.html` והעלה מחדש.
- לצפייה/ניקוי ביקורות — פתח ישירות את ה-Google Sheet.
